# VS Code Extensions

Extensions used for working on this repo from VS Code.

## Python

| Extension | Publisher | ID | Link |
| --- | --- | --- | --- |
| Python | Microsoft | `ms-python.python` | [marketplace](https://marketplace.visualstudio.com/items?itemName=ms-python.python) |
| Pylance | Microsoft | `ms-python.vscode-pylance` | [marketplace](https://marketplace.visualstudio.com/items?itemName=ms-python.vscode-pylance) |
| Python Debugger | Microsoft | `ms-python.debugpy` | [marketplace](https://marketplace.visualstudio.com/items?itemName=ms-python.debugpy) |
| Jupyter | Microsoft | `ms-toolsai.jupyter` | [marketplace](https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter) |

The Python extension is the core tooling (interpreter selection, linting,
formatting, test discovery). Pylance is the language server (type
checking, IntelliSense). The Debugger is needed for `Run and Debug` to
work, and Jupyter enables `.ipynb` notebooks directly in the editor.

## Google Colab

| Extension | Publisher | ID | Link |
| --- | --- | --- | --- |
| Google Colab | Google | `googlecolab.colab` | [marketplace](https://marketplace.visualstudio.com/items?itemName=Google.colab) |

Lets you open notebooks from Google Drive, attach to a Colab runtime
(free, Pro, or Pro+ GPU/TPU), and run cells without leaving VS Code.
Pairs with the Jupyter extension above for keyboard shortcuts and the
notebook UI.
