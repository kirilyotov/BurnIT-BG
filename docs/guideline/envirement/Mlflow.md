# Install MLflow

1. You should already have python and activated virtual envirement
``` bash 
pip install mlflow
```

2. Run it loclly:
``` bash
mlflow server --port 5000
```

3. If you still do not have `.env` file create one you can use `.env-example` and change `MLFLOW_TRACKING_URI`
