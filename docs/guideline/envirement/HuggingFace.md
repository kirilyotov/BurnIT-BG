# Install HuggingFace

1. Install cli
```
pip install huggingface_hub
```

2. Generate token:
Login your profile in huggingface  and after that settings -> Access tokens or just click [here](https://huggingface.co/settings/tokens).

- generate write token and after that update `.env` file
- if you want to use hf cli then:
``` bash 
huggingface-cli login
```
