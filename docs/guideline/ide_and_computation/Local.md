# Local Development and model execution

For local development you will need to have installed [Python](../envirement/Python.md), and IDE which supports Jupiter notebooks - [VsCode](https://code.visualstudio.com/download) with [extentions](./VsCodeExtentions.md) or [PyCharm](https://www.jetbrains.com/pycharm/download/?section=linux). Also traing on CPU is not time efficient and it is good to have good GPU for training the models using [CUDA](https://blogs.nvidia.com/blog/what-is-cuda-2/) ( [how to install](../envirement/Cuda.md) ) and [Pytorch](../envirement/Pytorch.md).  

## Model monitoring
[Mlflow](https://mlflow.org/docs/latest/ml/) for monitoring training, hardware, results and  comparison between LLM models.

## Mlflow setup 
Local install on MLFlow [here](https://mlflow.org/docs/latest/ml/tracking/quickstart/)
